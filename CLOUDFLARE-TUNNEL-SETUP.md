# Cloudflare Tunnel Setup (sanitized)

This repo uses **Cloudflare Tunnel** to expose the Shahin AI domains while keeping service ports private.

**Do not commit secret values.** Use the secret name:

- `CLOUDFLARE_TUNNEL_TOKEN`
